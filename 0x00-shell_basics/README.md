# System Engineering & DevOps Repository

This repository contains all projects and scripts related to my learning journey in System Engineering and DevOps principles, focusing on shell scripting, permissions, redirections, and system administration tasks.

---

## Project Structure

* **0x01-shell_basics:** Shell scripts focusing on file/directory basics (`chmod`), ownership (`chown`, `chgrp`), and user/group management.
* **0x02-shell_redirections:** Shell scripts demonstrating input/output redirection, filtering, and special file descriptors.
